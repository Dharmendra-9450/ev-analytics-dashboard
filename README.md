# EV Analytics Dashboard 🚗⚡

This project analyzes Electric Vehicle charging patterns, battery performance, and model efficiency using data visualization techniques. The dashboards are created using Tableau and integrated into a web interface.

---

# 📊 Dataset

Source: Kaggle  
Dataset Name: Electric Vehicle Charge and Range Analysis

The dataset contains the following fields:

### Dimensions
- Charger Type
- Charging Station ID
- Charging Station Location
- Day of Week
- Time of Day
- User ID
- User Type
- Vehicle Model

### Measures
- Battery Capacity (kWh)
- Charging Cost (USD)
- Charging Duration (hours)
- Charging Rate (kW)
- Distance Driven
- Energy Consumed (kWh)
- State of Charge (Start %)
- State of Charge (End %)
- Temperature (°C)
- Vehicle Age (years)

### Time Fields
- Charging Start Time
- Charging End Time

# 📊 Dashboards

## 1️⃣ EV Charging Pattern & Demand Analysis

This dashboard analyzes EV charging usage patterns.

Visualizations included:

- Peak Charging Hours
- Day Wise Energy Usage
- Location Based Charging Demand
- Charger Type Distribution

---

## 2️⃣ Battery Performance & Charging Efficiency

This dashboard analyzes battery performance and charging behavior.

Visualizations included:

- Distance Driven vs Energy Consumed
- Temperature vs Energy Consumed
- SOC (Start %) vs Energy Consumed
- Charging Rate vs Charging Duration

---

## 3️⃣ EV Model Comparison Dashboard

This dashboard compares different EV models based on performance metrics.

Visualizations included:

- Average Efficiency
- Average Battery Capacity
- Average Charging Cost
- Average Driving Range
- Charger Type Distribution by Model

---

# 📖 Story

A Tableau Story is used to present the analysis step by step:

1. EV Charging Pattern Analysis
2. Battery Performance and Charging Behavior
3. EV Model Efficiency Comparison

The story helps explain the complete insights clearly.

---

# 🌐 Web Integration

The Tableau dashboards and story are embedded into a responsive website using HTML, CSS, and JavaScript.

Users can view and interact with the dashboards directly on the webpage.

# 📌 Key Insights

- Peak EV charging activity occurs during evening hours.
- Fast chargers reduce charging duration significantly.
- Energy consumption varies with driving distance.
- Some EV models provide better efficiency and range.
